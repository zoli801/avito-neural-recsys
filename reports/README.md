# Reports

Run `python scripts/eda.py` in the Kaggle notebook to generate the EDA plots:

- `reports/figures/items_by_vertical.png`
- `reports/figures/history_length.png`
- `reports/figures/event_id_frequency.png`
- `reports/figures/contacts_by_vertical.png`

The image files are generated from the official attached dataset and are intentionally not committed from the local machine.
